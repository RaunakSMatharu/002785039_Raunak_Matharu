The following are the tags that I have used in the Part A of Assignment 1. 

1. header tag 
The <header> tag in HTML is used to define the header for a document or a section as it contains the information related to the title and heading of the related content.

2. section
The <section> tag defines a section in a document.

3. image tag
The <img> tag is used to embed an image in an HTML page.

4. anchor tag
An anchor tag is a HTML element that creates a link to a target URL

5. button tag
The <button> tag defines a clickable button. Inside a <button> element you can put text .

6. nav tag
The <nav> tag defines a set of navigation links.

7. div tag
The div tag is known as Division tag. The div tag is used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc)

8. br tag
he <br> HTML element produces a line break in text 

9. form tag
The <form> element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

10. table tag
The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.

11. ul and li tags
The <ul> HTML element represents an unordered list of items, typically rendered as a bulleted list.
The <li> HTML element is used to represent an item in a list.

12. p tags
The <p> tag defines a paragraph. Browsers automatically add a single blank line before and after each <p> element

13. Heading tags
HTML defines six levels of headings. A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading.
The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the least.
